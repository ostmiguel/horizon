# Personal Budget App

Личное финансовое приложение на HTML + Supabase.

## Запуск

1. Клонируй репозиторий
2. Запусти локальный сервер
3. Открой в браузере: `http://localhost:8080/budget-v2.html`

4. Введи свои Supabase URL и Anon Key

## База данных

1. Создай проект на [supabase.com](https://supabase.com)
2. Открой SQL Editor и выполни `schema.sql`
3. URL и Anon Key найдёшь в Settings → API

## Стек

- HTML + Vanilla JS
- Supabase (PostgreSQL)
- Chart.js
